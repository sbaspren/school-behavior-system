## التنفيذ

### 1. فصل الأدمن عن المدير
**الملفات**: `DeputiesSection.tsx`, `constants.ts`

**التغييرات**:
- تغيير ليبل "المدير" → "الأدمن" في بطاقة الأدمن
- إضافة "مدير المدرسة" كدور في `ADMIN_ROLES` في `AdminsTab` بحيث يمكن تعيين شخص كمدير مدرسة مستقل عن الأدمن
- **لا** نحذف "مدير المدرسة" من `ADMIN_ROLES` — يبقى كدور قابل للتعيين لأي شخص

### 2. إصلاح mergeStages لتصحيح الفصول
**الملف**: `client/src/pages/SettingsPage.tsx`

**التغييرات في `mergeStages`**:
```
- لما يكون فيه بيانات محفوظة:
  1. يطابق الفصول المحفوظة مع الثوابت (بالاسم العربي أو الإنجليزي)
  2. يصحح أسماء الفصول الإنجليزية للعربية
  3. يضيف أي فصول ناقصة من SETTINGS_STAGES (disabled + classCount=0)
  4. يحافظ على الإعدادات المحفوظة (classCount, isEnabled) للفصول الموجودة
```

### 3. إضافة mapping إنجليزي → عربي
**الملف**: `client/src/pages/SettingsPage.tsx`
```ts
const ENGLISH_TO_ARABIC: Record<string, string> = {
  'first': 'الأول', 'second': 'الثاني', 'third': 'الثالث',
  'fourth': 'الرابع', 'fifth': 'الخامس', 'sixth': 'السادس',
};
```

### 4. تصحيح قاعدة البيانات
- تحديث أسماء الفصول الإنجليزية في `grade_configs` مباشرة
- إضافة الصفوف الناقصة للمراحل الموجودة

### 5. التحقق من زر إضافة وكيل
- الكود يبدو صحيح — الزر يفتح DeputyModal
- ربما المشكلة إن المراحل المفعلة فارغة فيظهر رسالة "لم يتم تفعيل أي مرحلة"
- يحتاج تحقق عملي بعد إصلاح الفصول
