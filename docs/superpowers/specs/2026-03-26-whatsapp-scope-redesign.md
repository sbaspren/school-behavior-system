# إعادة تصميم نظام الواتساب والربط بالمراحل

## الهدف
إعادة هيكلة نظام الواتساب ليدعم 4 سيناريوهات مختلفة، مع ربط كل مستخدم (معلم/موجه/إداري/حارس) بالوكيل المسؤول عن مرحلته.

## السيناريوهات الأربعة للواتساب

### السيناريو 1: مدير فقط بدون وكلاء
- Admin واحد فقط في النظام
- رقم واتساب واحد
- يتنقل بين المراحل كفلتر للطلاب
- يربط الرقم مرة واحدة بدون اختيار مرحلة
- يرسل ويطبع ويوثق لكل المراحل

### السيناريو 2: مدير + وكلاء — رقم واحد للمدرسة
- Admin يربط رقم واتساب واحد (الرقم الرئيسي)
- يوزع الوكلاء على المراحل
- الوكلاء يرسلون من رقم المدرسة الرئيسي
- الوكيل لا يربط رقم — فقط يرسل
- الوكيل يشوف فقط مرحلته/مراحله

### السيناريو 3: مدير + وكلاء — الوكيل مخيّر
- Admin يربط رقم رئيسي
- يوزع الوكلاء على المراحل
- كل وكيل يختار: رقم المدرسة أو رقمه الخاص
- إذا اختار رقمه: يربطه من واجهة الواتساب بـ QR

### السيناريو 4: بدون رقم رئيسي — كل وكيل مستقل
- Admin لا يربط رقم (إلا إذا أراد يرسل بنفسه)
- كل وكيل يربط رقمه الخاص بشكل مستقل
- كل وكيل يتعامل فقط مع مرحلته
- المدير يستعرض كل المراحل للمراقبة
- لا أحد يرسل بدون رقم مربوط

## القواعد العامة

### المرحلة = فلتر للطلاب
- المرحلة ليست مرتبطة بالرقم — هي فلتر لاختيار الطلاب
- حتى في النمط الموحد (رقم واحد)، المستخدم يختار المرحلة لتحديد الطلاب المستهدفين

### كل مستخدم مربوط بمرحلة من الإعدادات
- المدير: كل المراحل تلقائياً
- الباقي: حسب ما يحدده المدير في الإعدادات

### المدير فقط يشوف كل المراحل
- أي مستخدم آخر يشوف فقط المراحل المربوط فيها

## صلاحيات الأدوار

| الدور | يشوف صفحة الواتساب؟ | يربط رقم؟ | يرسل واتساب؟ | يُربط بمرحلة؟ |
|-------|---------------------|----------|-------------|--------------|
| المدير | نعم — كل المراحل | نعم | نعم | كل المراحل |
| الوكيل | نعم — مرحلته فقط | نعم أو يستخدم رقم المدرسة | نعم | حسب الإعدادات |
| المعلم | لا | لا | إذا فُعّلت الصلاحية (من الرقم المربوط) | حسب الإعدادات |
| الموجه | لا | لا | إذا فُعّلت الصلاحية (من الرقم المربوط) | حسب الإعدادات |
| الإداري | لا | لا | لا | حسب الإعدادات |
| الحارس | لا | لا | لا | حسب الإعدادات |

## سلسلة الربط

```
المدير ──→ يشوف الكل ويدير الكل
  ├── وكيل ابتدائي ──→ معلمين ابتدائي + موجه ابتدائي + إداري + حارس
  ├── وكيل متوسط   ──→ معلمين متوسط + موجه متوسط + إداري + حارس
  └── وكيل ثانوي    ──→ معلمين ثانوي + موجه ثانوي + إداري + حارس
```

- المعلم مربوط بفصول ← الفصول تتبع مرحلة ← المرحلة لها وكيل
- الموجه/الإداري/الحارس مربوط بمرحلة ← المرحلة لها وكيل
- المراسلات من المعلم/الموجه تظهر تلقائياً عند وكيل المرحلة

## إرسال الروابط للمعلمين

### طريقة 1: من المدير (الإعدادات)
- المدير يولّد روابط التوكن من LinksTab ويرسلها مباشرة

### طريقة 2: من الوكيل (واجهة الواتساب)
- الوكيل يشوف فقط المعلمين المربوطين بمرحلته
- يرسل لهم الروابط من رقمه الخاص عبر الواتساب
